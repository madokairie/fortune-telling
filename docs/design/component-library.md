## 5. 和モダン装飾の適用ルール

### 5.1 適用する場面

| 場面 | 装飾方法 | 注意 |
|------|---------|------|
| **セクション区切り** | 細い和の罫線（opacity: 0.1） | Separatorの代替 |
| **ダッシュボード背景** | 淡い星座線パターン（opacity: 0.03〜0.06） | CSSのbackground-image |
| **ページ遷移** | フェードイン（opacity変化のみ） | 200ms以内 |
| **カード装飾** | 左ボーダーに金のアクセントライン | border-left: 2px solid |
| **見出し装飾** | 下線に和の模様（点線パターン） | border-bottom: dotted |

### 5.2 装飾の濃度ルール

| 要素 | 最大opacity | 備考 |
|------|-----------|------|
| 背景パターン | 0.06 | 読みやすさを最優先 |
| 装飾線 | 0.15 | コンテンツの邪魔をしない |
| ボーダーアクセント | 1.0 | Soft Goldのみ |
| アイコン装飾 | 0.08 | 背景に溶け込む程度 |

### 5.3 装飾の適用頻度

- **1画面に装飾要素は最大3つ**
- **すべてが主張しない**（控えめが原則）
- 迷ったら**装飾しない**を選ぶ
