## 1. プロジェクト概要

### 1.1 プロダクトビジョン

紫微斗数・方位術・その他東洋占術の専門知識とAIを融合し、円香（まどか）の事業発展とプライベート充実を日常的に支援するパーソナルアプリケーション。

### 1.2 ユーザープロフィール

| 項目 | 内容 |
|------|------|
| 氏名 | 円香（まどか） |
| 年齢・性別 | 44歳女性 |
| 家族構成 | バツイチ独身、子供なし |
| 事業1 | **株式会社エムクリエイト** — SNSオンラインコンテンツ販売支援コンサル・講座運営 |
| 事業2 | **JYビー協会** — 陰陽五行美容ベースの体質改善サポート |
| 役職 | 両社代表（法人2社経営） |

### 1.3 プロジェクト目的

1. プロの鑑定士から受けた紫微斗数・方位術等のアドバイスをデジタルナレッジとして永続化
2. AIを活用し、命盤データ＋プロのナレッジに基づく的確なアドバイスを随時取得可能に
3. 日運・月運・年運に基づく最適な行動タイミングの提示
4. 事業判断（エムクリエイト・JYビー協会）とプライベート（旅行・健康・パートナーシップ）の両面をサポート

## 2. 機能要件

### Phase 1（コア機能）

#### FR-001: AI相談チャット

- **優先度**: Critical
- **説明**: 円香の命盤＋プロ鑑定ナレッジをベースに、事業・プライベートの相談に的確に回答するAIチャット機能
- **詳細**:
  - Claude APIを使用したチャットインターフェース
  - 円香の紫微斗数命盤データをシステムプロンプトに常時保持
  - プロ鑑定士のアドバイス（文字起こし＋鑑定書類）をRAGナレッジとして検索・参照
  - 質問のコンテキスト（事業/プライベート、エムクリエイト/JYビー協会）に応じた回答
  - 会話履歴の保持（セッション内）
  - 過去の会話履歴の保存・検索
- **受入基準**:
  - [ ] チャットUIで質問を入力し、AIから回答が返る
  - [ ] 命盤データに基づいた回答が得られる
  - [ ] プロ鑑定士のアドバイスが適切に引用・参照される
  - [ ] 事業面（エムクリエイト/JYビー協会）の相談に対応
  - [ ] プライベート面（旅行/健康/パートナーシップ）の相談に対応
  - [ ] 会話履歴が保存される

#### FR-002: 今日のアドバイス

- **優先度**: Critical
- **説明**: 日運に基づく毎日のアドバイスをダッシュボードに表示
- **詳細**:
  - 当日の干支・流日星の配置に基づくアドバイス生成
  - 仕事運（エムクリエイト向け・JYビー協会向け）
  - プライベート運（人間関係・健康・パートナーシップ）
  - 今日の吉方位
  - 今日のラッキーカラー・ラッキーアイテム（陰陽五行ベース）
  - アプリ起動時に自動表示
- **受入基準**:
  - [ ] アプリ起動時に当日のアドバイスが表示される
  - [ ] 仕事とプライベートの両面でアドバイスが提供される
  - [ ] 毎日異なるアドバイスが生成される
  - [ ] 日付が変わると新しいアドバイスに更新される

#### FR-003: 基本性質フィードバック

- **優先度**: High
- **説明**: 円香の命盤に基づく基本性質を踏まえた行動アドバイス
- **詳細**:
  - 紫微斗数の命盤から読み取れる基本性質の提示
  - 特定の状況（新規事業判断、人間関係、パートナーシップ等）における性質を活かしたアドバイス
  - プロ鑑定士のアドバイスと連動した性質解説
  - 「あなたの性質的にこうするといい」形式のフィードバック
- **受入基準**:
  - [ ] 基本性質のサマリーページが表示される
  - [ ] 相談時に性質に基づいたアドバイスが含まれる
  - [ ] プロ鑑定士のアドバイスと整合性がある

### Phase 2（拡張機能）

#### FR-004: 吉日・吉月カレンダー

- **優先度**: Medium
- **説明**: 事業判断・プライベート行動の最適タイミングを視覚的に提示
- **詳細**:
  - 月間カレンダービュー
  - 吉日・凶日のカラー表示
  - 日ごとの詳細（吉時間帯、避けるべき行動等）
  - 事業イベント（講座開催、コンサル、新サービスローンチ）の最適日提案
  - プロ鑑定士の年間データとの連動
- **受入基準**:
  - [ ] カレンダー形式で吉日・凶日が視覚的に確認できる
  - [ ] 日付タップで詳細情報が表示される
  - [ ] 月別の全体運勢が表示される

#### FR-005: 吉方位マップ

- **優先度**: Medium
- **説明**: 旅行先・移動の方位アドバイスを視覚的に提示
- **詳細**:
  - 八方位の吉凶表示
  - 月別・年別の吉方位データ（プロ鑑定士の年間データに基づく）
  - 旅行先の方位チェック機能
  - 地図上での吉方位エリア表示
- **受入基準**:
  - [ ] 八方位の吉凶が視覚的に確認できる
  - [ ] 月別の吉方位が切り替え表示される
  - [ ] 目的地を入力すると方位の吉凶が判定される

#### FR-006: 風水アドバイス

- **優先度**: Low
- **説明**: 自宅・オフィスの風水に関する助言
- **詳細**:
  - 円香の命盤に基づく風水のポイント
  - 部屋のレイアウトアドバイス
  - AI相談チャットでの風水相談対応
- **受入基準**:
  - [ ] 風水に関する基本アドバイスが閲覧できる
  - [ ] チャットで風水の質問に回答が得られる

#### FR-007: 過去アドバイス検索

- **優先度**: Low
- **説明**: これまでのAI相談履歴とアドバイスの検索
- **詳細**:
  - 過去の会話をキーワード検索
  - カテゴリ別（事業/プライベート）フィルター
  - 日付範囲指定検索
  - ブックマーク機能（重要なアドバイスを保存）
- **受入基準**:
  - [ ] 過去の会話が検索できる
  - [ ] カテゴリ・日付でフィルタリングできる
  - [ ] 重要なアドバイスをブックマークできる

## 10. ユーザーストーリー

### US-001: 毎朝のアドバイス確認

> 円香として、アプリを開いたら今日の運勢とアドバイスを確認したい。それにより、今日の仕事の進め方とプライベートの過ごし方を最適化できる。

### US-002: 事業判断の相談

> 円香として、「新しいクライアントからコンサル依頼が来たが受けるべきか」をAIに相談したい。命盤の性質とプロのアドバイスに基づいた判断材料が欲しい。

### US-003: 旅行先の方位確認

> 円香として、旅行を計画する際に行き先の方位が良いか確認したい。今月の吉方位と照合して最適な旅行先を選びたい。

### US-004: 忘れたアドバイスの確認

> 円香として、以前プロの鑑定士にもらったアドバイスを思い出したい。キーワードで検索して、関連する鑑定内容を確認したい。

### US-005: 講座開催日の選定

> 円香として、来月の講座開催日を決めるために吉日カレンダーを確認したい。事業運の良い日に講座を設定したい。

### US-006: パートナーシップの相談

> 円香として、パートナーシップについてAIに相談したい。命盤から読み取れる傾向と、今の時期のアドバイスが欲しい。

## 12. リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| AI応答の精度不足 | アドバイスの信頼性低下 | システムプロンプトの継続的改善、プロナレッジの充実 |
| Claude API の料金増加 | ランニングコスト増 | キャッシュ戦略、回答の再利用 |
| ナレッジデータの不足 | 回答の幅が狭い | 鑑定資料の継続的追加 |
| 紫微斗数の計算精度 | 日運アドバイスの正確性 | プロ鑑定データとの照合、理論書の参照 |
